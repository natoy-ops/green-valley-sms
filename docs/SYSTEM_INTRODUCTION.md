### Core Capabilities

**1. User & Access Management**
- Bulk import staff, teachers, students with one Excel file
- Automatic secure password generation
- Role-based access (Admin, Teacher, Staff, Parent, Student)
- Complete audit trail of all actions

**2. Student Information System**
- Import students with guardian information simultaneously
- Auto-link siblings to same parent account
- Unique QR code generated for every student
- Export QR codes for ID cards (Excel or printable format)

**3. Facility & Venue Management**
- Register all school facilities with capacity tracking
- Real-time availability checking
- Automatic conflict prevention for event bookings

**4. Event Management**
- Create events in minutes
- Invite by grade, section, or individual
- RSVP collection with live response tracking
- Send reminders to non-responders with one click

**5. QR Attendance Scanning**
- Scan students in under 2 seconds
- Works on any device with a camera
- Real-time attendance dashboard
- Automatic late arrival flagging
- Duplicate scan prevention
