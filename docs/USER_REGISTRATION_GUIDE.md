## Overview

The School Management System uses an **administrator-managed registration model**. This means:

> **Important:** There is no public self-registration. All user accounts are created by authorized administrators.

This approach ensures:
- Only verified individuals gain access to the system
- Proper role assignment from the start
- Secure credential distribution
- Complete audit trail of account creation

---

## User Roles in the System

The system supports **seven distinct user roles**, each with specific access levels:

| Role | Icon | Description | Who Creates Them |
|------|------|-------------|------------------|
| **Super Admin** | `[SA]` | Full system access and configuration | System Setup |
| **Admin** | `[A]` | School administration and user management | Super Admin |
| **Teacher** | `[T]` | Teaching staff with class management | Admin |
| **Staff** | `[S]` | General administrative support | Admin |
| **Student** | `[ST]` | Student portal access | Admin (via import) |
| **Parent** | `[P]` | Guardian access to student information | Admin |
| **Scanner** | `[SC]` | Event attendance scanning (kiosks) | Admin |

### Detailed Steps

#### **Step 1: Navigate to User Management**
1. Log in to the system as an Administrator
2. From the main menu, click on **"Users"** or **"User Management"**
3. The user list page will display

---

#### **Step 2: Click "Add User" Button**
1. Locate the **"+ Add User"** button (usually at the top right)
2. Click the button
3. A dialog form will appear

---

#### **Step 3: Fill in User Details**

| Field | What to Enter | Example |
|-------|---------------|---------|
| **Full Name** | Complete name of the user | Maria Santos Cruz |
| **Email Address** | Valid email (will be their login) | maria.cruz@school.edu |
| **Role** | Select from dropdown | Teacher |

> **Note:** The system will automatically generate a secure temporary password.

---

#### **Step 4: Click "Create User"**
1. Review the information entered
2. Click the **"Create User"** button
3. Wait for confirmation

---

#### **Step 5: Copy the Temporary Password**
1. A success message will appear with the temporary password
2. **IMPORTANT:** Copy this password immediately
3. The password will NOT be shown again after closing
