### Key Concepts to Understand

| Concept | Explanation |
|---------|-------------|
| **Event Name** | Title of the event (e.g., "PTA Meeting - December 2025") |
| **Event Type** | Category of the event (e.g., Meeting, Program, Activity) |
| **Venue** | Facility/location where the event will be held |
| **Participants** | Students, parents, or staff invited to the event |
| **RSVP** | Response from invitees indicating their attendance intention |
| **Attendance** | Actual record of who attended on event day |
| **Status** | Current state: Draft, Published, Ongoing, Completed, Canceled |
